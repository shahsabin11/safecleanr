# Student Performance Dataset

A sample dataset containing student academic information, including
study hours, attendance, and final grades.

## Usage

``` r
student_data
```

## Format

A data frame with 8 rows and 6 variables:

- student_id:

  Unique student ID

- name:

  Student name

- major:

  Student major

- study_hours:

  Weekly study hours

- attendance:

  Attendance percentage

- final_grade:

  Final course grade

## Source

Simulated dataset for package demonstration.
